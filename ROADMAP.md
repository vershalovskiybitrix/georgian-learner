# Georgian Learner — Roadmap

## V1.0 — Транслитерация (текущая версия)

- Замена X букв из 1000 в обычных текстах страниц
- Поддержка латиницы и кириллицы
- Три режима приоритета мультисимвольных паттернов
- Визуальная подсветка заменённых букв (опционально)
- MutationObserver для динамического контента (соцсети, SPA)
- Чёрный/белый список доменов
- Страница настроек с редактируемой таблицей соответствий

---

## V1.x — Улучшения транслитерации

### Расширение таблиц соответствий
- Добавить поддержку других исходных языков (немецкий, французский, испанский → грузинский)
- Добавить поддержку других изучаемых языков (армянский, арабский, хинди и т.д.)
- Формат JSON уже готов к расширению — поля `source` и `target`

### Защита от коллизий (простой уровень)
- Не заменять буквы внутри слов-исключений (загружаемый список)
- Не заменять мультисимвольные паттерны на границах слов по эвристике (без полноценного NLP)

### Контекстная подсказка
- Hover над заменённым символом → тултип с оригиналом и транскрипцией
- Клик → небольшой popup с информацией о звуке

### Статистика
- Счётчик «встреченных» грузинских символов за сессию
- Показывать в popup: «сегодня встретил X разных символов»

## V2.0 — Перевод слов

### Архитектура
Уже заложена: `WordTranslationStrategy` реализует тот же интерфейс что и `LetterReplacementStrategy`.

### Функциональность
- Замена отдельных слов на грузинские эквиваленты (не транслитерация, а перевод)
- Локальный словарь (бандл в расширении), без сетевых запросов в базовом режиме
- Уровни: начинающий (100 слов), базовый (500), продвинутый (2000+)
- Hover → оригинальное слово + произношение

### Словарь наиболее употребительных слов
**Ключевая идея:** учить не весь язык, а топ-1000 самых частотных слов — это покрывает ~80% обычного текста. Загрузить из открытых частотных словарей (Wiktionary, CorpusGeorgian), отобрать топ по частоте, упаковать в бандл. Пользователь быстро видит прогресс, потому что освоенные слова встречаются на каждой странице.

### Смешанный режим
- Одновременно: часть букв транслитерируется, часть слов переводится
- Настройка: какой процент — транслитерация, какой — перевод

## V2.x — Улучшения словарного режима

### Уровни сложности
- Автоматический подбор слов по истории встреч: слово переводится, когда пользователь уже видел его 10+ раз
- «Выучено» — слово снова показывается в переведённом виде реже

### Активное повторение
- Необязательный режим: иногда показывать popup «как переводится это слово?» при наведении

### Сетевой режим (опциональный)
- Подключение к Georgian WordNet или Wiktionary API для расширенного словаря
- Только по явному запросу пользователя, с уведомлением о сетевом запросе

## Технический долг и известные проблемы к решению

### Коллизии транслитерации (запланировано на V2)
Английское `sh` может быть как звук «ш» (ship → შიp), так и два звука на стыке морфем (mishap → mis-hap). Для правильной обработки нужен морфологический анализатор. В V1 принимаем как ограничение, в V2 решаем через словарь исключений или NLP.

### Производительность на тяжёлых SPA
MutationObserver с дебаунсом 250ms достаточно для V1. Если возникнут жалобы на лаги — переход на `requestIdleCallback` с батчингом узлов, или ограничение глубины обхода DOM.

### Иконочные шрифты
Material Icons, Font Awesome и подобные используют буквы как коды символов. Whitelist родительских элементов снижает риск, но не исключает 100%. В V2 добавить детектирование шрифтов по `font-family` computed style.

### Лимиты chrome.storage.sync
Настройки (rate, enabled, etc.) — малые, в sync. Кастомные маппинги — в local, soft limit 10 КБ. При росте словаря V2 перенести всё в IndexedDB.

## Идеи для исследования

- **Частотный словарь грузинского**: топ-500 слов покрывают ~70% обычного текста. Источники: Georgian National Corpus, Leipzig Corpora, Wiktionary frequency lists.
- **Адаптивная частота**: плагин сам снижает rate для уже хорошо знакомых символов и повышает для редко встречаемых.
- **Режим «только заголовки»**: заменять буквы только в `h1–h3` — максимально незаметно, но регулярно.
- **Расширение на другие редкие алфавиты**: армянский, грузинские диалекты, старогрузинский (Асомтаврули).
